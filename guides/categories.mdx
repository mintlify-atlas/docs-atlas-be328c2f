## Accessing categories

<Steps>
  <Step title="Open categories page">
    Click the grid icon in the top navigation bar or visit `/categories` to see all available categories.
  </Step>
  
  <Step title="Browse category cards">
    Each category displays:
    - Representative wallpaper as background
    - Category icon
    - Category name
    - Brief description
  </Step>
  
  <Step title="Select a category">
    Click any category card to view all wallpapers in that category.
  </Step>
</Steps>

## Category page features

When viewing a specific category:

### Header section
- Large category icon and name
- Descriptive text explaining the category theme
- Back button to return to all categories

### Wallpaper grid
All standard browsing features are available:
- Layout switching (masonry/grid)
- Device filtering (all/desktop/mobile)
- Infinite scroll loading
- Hover/tap actions (download, favorite, share, expand)

### Navigation
Use the back button in the header to return to the categories overview page.

## Category card interactions

### Desktop experience
- **Hover effects**: Cards scale up slightly (1.02x) with smooth transitions
- **Background zoom**: The preview image zooms in (1.1x) on hover
- **Enhanced details**: Icon scales up (1.25x) and divider line expands
- **Border highlight**: White border appears on hover

### Mobile experience
- **Tap to navigate**: Simply tap any category card
- **Optimized animations**: Faster animations (200ms) for immediate feedback
- **Staggered loading**: Cards animate in with delays (100ms between each)

<Tip>
  Category preview images are loaded dynamically from the first wallpaper in each category, giving you a real preview of what to expect.
</Tip>

## Category metadata

Categories are stored in the wallpaper data with the format `#categoryname`. For example:
- `#nature` for Nature wallpapers
- `#anime` for Anime wallpapers
- `#minimal` for Minimal wallpapers

This system allows for efficient filtering and organization.

## Finding categories in search

You can also search for wallpapers by category name using the search bar. The search function includes category tags in its results.

<Note>
  Categories are continuously updated as new wallpapers are added to the platform. Check back regularly for fresh content in your favorite categories.
</Note>
